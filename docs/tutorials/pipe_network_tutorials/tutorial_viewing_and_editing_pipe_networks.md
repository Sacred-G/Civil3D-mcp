# Tutorial: Viewing and Editing Pipe Networks

This tutorial demonstrates how you can view and edit the parts of your pipe network in profile and section views.

You can draw some or all of the parts of a pipe network in a profile view. After the parts are drawn in a view, you can adjust their vertical layout by either grip editing or directly editing the values found in tabular format in the part properties dialog box. You also can add tables to organize pipe network part data, and label the parts to make them easy to identify.

When you create a section view, any pipe network parts that are located at the cross-section are drawn in the view.
