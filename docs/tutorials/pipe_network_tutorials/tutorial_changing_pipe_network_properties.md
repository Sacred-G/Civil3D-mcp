# Tutorial: Changing Pipe Network Properties

This tutorial demonstrates how to add parts to your pipe network parts list. You will also learn how to change the surface, alignment, and design rules that are referenced when you are laying out a pipe network.

In the previous tutorial, the storm system parts you used were designed for a carriageway design and took their placement and level from the road surface. In this tutorial, you create a branch for the pipe network that travels off the road surface into unfinished terrain, using manholes to connect the pipes and terminating in a headwall.

In addition to selecting different structures, pipes, and layout modes, you can also use the Network Layout Tools toolbar to change the configuration of your pipe network as you work. As you create your design, you can switch to a different parts list, add parts to the parts list, or change the referenced surface and alignment.

*   [Exercise 1: Adding Parts to the Parts List](GUID-089176F6-E1AC-41F4-96CD-B2AEABB5176C.htm)  
    In this exercise, you will add a new part to the parts list by selecting a part family and size from the pipe network part catalog.
*   [Exercise 2: Changing the Surface, Alignment and Rules Configuration](GUID-6FB5AA3B-B869-4CC5-A21A-A48F48149506.htm)  
    In this exercise, you will change the surface and alignment that are referenced by the pipe network parts. You will also examine the design rules for a part.
*   [Exercise 3: Adding a Branch to a Pipe Network](GUID-16A23062-0F4F-41BE-9ABE-8DABE9396FF0.htm)  
    In this exercise, you will add a branch to the existing pipe network layout and use the part status to review and edit the layout.
