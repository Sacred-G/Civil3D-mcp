# Tutorial: Creating a Divided Highway Corridor

This tutorial demonstrates how to create a divided highway corridor. The tutorial uses some of the subassemblies that are shipped with Autodesk Civil 3D to create a more complex and realistic highway model.

This highway has a depressed central reserve with a flat bottom ditch and inside paved shoulders, two travel lanes on each side of the central reserve, and paved outside shoulders. The total central reserve width between inside edges of carriageways is 44 feet. The highway may have at-gradient junctions, so the superelevation rotation point is set to be located at the design profile gradient above the centerline of the central reserve ditch. This creates a single consistent crossfall across the entire carriageway in superelevated areas.
