# Exercise 5: Editing Points

In this exercise, you will move and rotate point objects to improve their position in the drawing.

Change the point style

Note:

This exercise uses Points-3.dwg, which is similar to the drawing you used in the other exercises in the [Displaying and Editing Points](GUID-E43EE235-038D-49EB-8307-6E8D00A9B3A4.htm "This tutorial demonstrates how to use point groups, layers, external references, and styles to display points. It also explains the various ways to edit points using standard AutoCAD tools.") tutorial.

2.  Open Points-3.dwg, which is located in the [tutorials drawings folder](GUID-DECD2305-4906-4329-A973-CFC9B625B4CD.htm#GUID-DECD2305-4906-4329-A973-CFC9B625B4CD__WS27101FEF35E777F9922804FBF5414E08-8000-79).
3.  Locate point 992 near the left side of the drawing.
    
    Notice that the label for point 992 is obscured by the hatching. In the next few steps, you will change the point marker style. You will rotate the point marker so that it is parallel with the nearby road, and then move the label to a more readable location.
    
4.  Select point **992**. Right-click. Click Properties. The AutoCAD Properties palette displays the properties of individual points.
5.  Under Information, select **Gully** in the Style property.

Rotate the point

1.  In the drawing window, hover the cursor over the ![](../images/GUID-4828FA35-B597-42FF-B9DF-4EC56AB4145A.png) grip.
    
    A menu of options is displayed. These options are related to the point marker, as well as the point object, which consists of both the point marker and the point label.
    
    Tip:
    
    To rotate a group of points, use the AutoCAD ROTATE command. To rotate a point marker or point label, you can enter rotation angle on the Properties palette.
    
2.  Click Rotate Label and Marker.
3.  Rotate the point clockwise until the point marker is parallel to the edge of the road, then click to position the point.
    
    The point marker now matches the angle of the road, but the label is still obscured by the hatching. You will correct this in the following steps.
    

Drag the point label

1.  Hover the cursor over the ![](../images/GUID-A8BAA3E4-DA9B-4F42-B645-32C522667DD5.png) grip.
    
    A menu of options is displayed. These options are related to the point label.
    
2.  Click Toggle Sub Item Grips.
    
    This toggles the display of the drag label grip. Three ![](../images/GUID-4828FA35-B597-42FF-B9DF-4EC56AB4145A.png) grips are available. These grips can be used to drag each line in the point label independently.
    
3.  Hover the cursor over the ![](../images/GUID-A8BAA3E4-DA9B-4F42-B645-32C522667DD5.png) grip.
4.  Click Toggle Sub Item Grips.
5.  Hover the cursor over the ![](../images/GUID-A8BAA3E4-DA9B-4F42-B645-32C522667DD5.png) grip.
6.  Click Move Label.
7.  Drag the label down and to the left, until it is in an unobstructed area. Click to place the label in its new orientation.
