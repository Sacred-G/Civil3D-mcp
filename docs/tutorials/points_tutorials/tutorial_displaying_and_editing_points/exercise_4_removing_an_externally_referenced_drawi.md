# Exercise 4: Removing an Externally Referenced Drawing

In this exercise, you will remove the externally referenced drawing that you added previously.

This exercise continues from [Exercise 3: Changing Point Group Display Order](GUID-3CC8E6EF-660E-4321-B499-5DFEB1E902C8.htm "In this exercise, you will use the point group display order to change the appearance of points.").

Remove an externally referenced drawing

Note:

This exercise uses Points-2.dwg and Existing Basemap.dwg with the modifications you made in the previous exercise.

2.  Click any element in Existing Basemap.dwg to select the whole drawing.
3.  Right-click, and click Xref Manager.
4.  In the External References dialog box, right-click the reference name **Existing Basemap**. Click Detach.
    
    The reference drawing disappears from view.
    
5.  Close the External References dialog box.
