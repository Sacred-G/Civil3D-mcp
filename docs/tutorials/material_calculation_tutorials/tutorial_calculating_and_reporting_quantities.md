# Tutorial: Calculating and Reporting Quantities

In this tutorial, you will learn how to create and manage rate item data, associate rate item codes with several types of drawing objects, and generate rate item quantity reports.

A _rate item_ is a specific unit of work for which a price is provided and paid to a contractor while a site is under construction. It consists of a rate item ID number, description, and unit of measure.

Rate items can be associated with any AutoCAD element, such as lines, closed polygons, and blocks, after they have been created. When rate items are associated with an Autodesk Civil 3D pipe network parts list or code set style, then the pipe network or corridor object is automatically tagged with the specified rate items.
