# Exercise 2: Adding a Section View Gradient Label

In this exercise, you will create a section view gradient label.

This exercise continues from [Exercise 1: Projecting an Object onto a Section View](GUID-0F40AF94-E87A-424A-A3F1-A619AA940590.htm "In this exercise, you will project multi-view blocks and 3D polylines from plan view onto section views.").

Add a section view gradient label

1.  Open Sections-Grade-Label.dwg, which is located in the [tutorials drawings folder](GUID-DECD2305-4906-4329-A973-CFC9B625B4CD.htm#GUID-DECD2305-4906-4329-A973-CFC9B625B4CD__WS27101FEF35E777F9922804FBF5414E08-8000-79).
2.  Click Annotate tab ![](../images/ac.menuaro.gif)Labels & Tables panel ![](../images/ac.menuaro.gif)Add Labels menu ![](../images/ac.menuaro.gif)Section View![](../images/ac.menuaro.gif)Add Section View Labels![](../images/GUID-10094573-05A5-439E-A30D-2E506775AA08.png).
    
    The Add Labels dialog box is displayed. You will change the existing Gradient Label Style to show a gradient percentage between two points and the gradient length between those points.
    
3.  In the Add Labels dialog box, specify the following parameters:
    *   Label Type: **Gradient**
    *   Gradient Label Style: **Gradient\_Slope**
4.  Click Add.
    
    On the command line, you are prompted to select a section view.
    
5.  In the drawing, click a grid line of the section view.
    
    On the command line, you are prompted to pick a point.
    
6.  In the section view, snap to a low point in the section. Snap to a higher point “up the hill”.
    
    The gradient between the two points is displayed. If you wish, add more gradient labels to other pairs of points anywhere in the section view.
    
7.  Right-click or press Esc to end the labeling session.
8.  In the Add Labels dialog box, click Close.
