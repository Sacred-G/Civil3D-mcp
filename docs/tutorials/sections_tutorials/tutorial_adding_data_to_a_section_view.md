# Tutorial: Adding Data to a Section View

This tutorial demonstrates how to add annotative data to a section view.

**Topics in this section**

*   [Exercise 1: Projecting an Object onto a Section View](GUID-0F40AF94-E87A-424A-A3F1-A619AA940590.htm)  
    In this exercise, you will project multi-view blocks and 3D polylines from plan view onto section views.
*   [Exercise 2: Adding a Section View Gradient Label](GUID-21890D53-6A5B-427F-88A0-EBD61C712EC2.htm)  
    In this exercise, you will create a section view gradient label.
*   [Exercise 3: Adding a Data Box to a Section View](GUID-126EF350-2364-4621-8F7E-3387174B95DC.htm)  
    In this exercise, you will add a data box, which is an optional graphic frame that is associated with the section view.
