# Tutorial: Importing Survey Data

This tutorial demonstrates how to import survey data into a drawing, modify the data, and then reprocess the data.

You will input survey data from a field book file, which is a text file that contains survey observation measurements. Typically, you download data from a data collector, and then create a field book file to import.

**Topics in this section**

*   [Exercise 1: Importing Field-Coded Survey Data](GUID-344B584B-0E65-4FCF-B85F-A62E2ABCD5DE.htm)  
    In this exercise, you will import survey data from an existing field book file that contains linework codes that can be interpreted by a linework code set.
*   [Exercise 2: Updating Imported Survey Data](GUID-23DF4893-016D-4ECF-9086-B8B4A6946DF3.htm)  
    In this exercise, you will modify some of the imported survey data, and then reprocess the linework to apply the changes.
