# Tutorial: Outputting Survey Information

This tutorial demonstrates how to view information reports for figures and how to use the figures as a source for surface data.

**Topics in this section**

*   [Exercise 1: Viewing Inverse and Mapcheck Information on a Survey Figure](GUID-5D6ECB87-49AD-4B66-97F0-39D14C25B128.htm)  
    In this exercise, you will display the figure mapcheck and inverse information.
*   [Exercise 2: Performing a Mapcheck Analysis with Plot Labels](GUID-712D93AB-2491-4C35-9FAE-206F56FC2F3F.htm)  
    In this exercise, you will use the data in plot segment labels to perform a mapcheck analysis.
*   [Exercise 3: Performing a Mapcheck Analysis by Manually Entering Data](GUID-234E732B-9B17-411C-A9AD-708E43A9826D.htm)  
    In this exercise, you will manually enter survey data to perform a mapcheck analysis.
*   [Exercise 4: Working with Mapcheck Data](GUID-8AB0AD8F-8AF4-47A5-919D-7E17860DB861.htm)  
    In this exercise, you will learn about the tools that can leverage the data obtained from a mapcheck analysis.
*   [Exercise 5: Creating Surface Breaklines from Figures](GUID-86A6B027-6D7D-4411-8930-DB437C0973A3.htm)  
    In this exercise, you will use figures to add breaklines to a surface.
