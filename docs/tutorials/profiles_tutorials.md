                 Profiles Tutorials  

# Profiles Tutorials

These tutorials will get you started viewing and designing the level profile of land surfaces along a horizontal alignment.

Note:

All drawings used in these tutorials are available in the [tutorials drawings folder](GUID-DECD2305-4906-4329-A973-CFC9B625B4CD.htm#GUID-DECD2305-4906-4329-A973-CFC9B625B4CD__WS27101FEF35E777F9922804FBF5414E08-8000-79). If you want to save your work from these tutorials, save the drawings to the [My Tutorial Data folder](GUID-DECD2305-4906-4329-A973-CFC9B625B4CD.htm#GUID-DECD2305-4906-4329-A973-CFC9B625B4CD__WS1A9193826455F5FF1848157110156D8271-7832) so that you do not overwrite the original drawings.

**Topics in this section**

*   [Tutorial: Designing Simple Profiles](GUID-0937FE14-98C7-4B24-BA8C-0822F31E280A.htm)  
    In this tutorial, you will create simple existing ground and layout profiles for an alignment.
*   [Tutorial: Using Surface Profiles](GUID-8B3EB320-E48D-4CB7-BD4D-AB887F1CF9B3.htm)  
    This tutorial demonstrates how to create surface profiles and display them in a profile view.
*   [Tutorial: Using Layout Profiles](GUID-DF73E4FC-35CB-4D11-96C9-8A73DB24DC45.htm)  
    This tutorial demonstrates how to create and edit layout profiles, which are often called design profiles or finished design profiles.
*   [Tutorial: Designing a Profile that Refers to Local Standards](GUID-75B9BF20-42D1-4D86-9465-C906A03FDC16.htm)  
    This tutorial demonstrates how to validate that your profile design meets criteria specified by a local agency.
*   [Tutorial: Displaying and Modifying Profile Views](GUID-0BF95BEA-BDFF-4B0A-A73C-6749A5FFD1C5.htm)  
    This tutorial demonstrates how to change the appearance of profile views.
*   [Tutorial: Working with Data Boxes](GUID-9CC02D3F-84C0-4574-89F0-61FBB809D683.htm)  
    This tutorial demonstrates how to add and change the appearance of data boxes in a profile view.
