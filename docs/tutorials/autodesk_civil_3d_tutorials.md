# Autodesk Civil 3D Tutorials

The following tutorial sets are included with Autodesk Civil 3D 2025:

*   [Getting Started Tutorials](GUID-82F849D8-FE3B-40B3-BE36-931D57B2DE4E.htm "These tutorials will get you started with the application workspace and some important design tools and tasks."). Learn about the application workspace and some important design tools and tasks.
*   [Points Tutorials](GUID-7EDF7EC4-829A-45B2-970F-AB559275E077.htm). Learn how to work with coordinate geometry (COGO) points, which are the basis for modeling land surfaces.
*   [Surfaces Tutorials](GUID-F6A629B7-EB6B-443A-91EE-491076F71440.htm). Learn the basics about creating and working with land surfaces.
*   [Survey Tutorials](GUID-20BCCB9B-BA7A-46E4-9ADF-9FF0D68083DC.htm "These tutorials will get you started working with the Autodesk Civil 3D survey features."). Learn the basics about importing, creating, managing, and analyzing survey data.
*   [Project Management Tutorials](GUID-31E17435-0D68-4ACE-A4AB-4ECCA0B31658.htm). Learn how to use the Autodesk Civil 3D project management features, including data shortcuts and Autodesk Vault.
*   [Alignments Tutorials](GUID-D625ABC3-2224-425A-BF5C-971439403C30.htm). Learn about alignments, which are the basis for modeling roads.
*   [Profiles Tutorials](GUID-768C10C5-7C9F-44F0-BDF2-4156801A41B2.htm). Learn the basics about viewing and designing the level profile of land surfaces along an alignment.
*   [Parcels Tutorials](GUID-B7C2A5F5-B258-44FD-8D3C-764AE8EE923D.htm). Learn the basics about creating and editing plots as well as working with the display of plots.
*   [Grading Tutorials](GUID-8144BA2B-9B57-41CC-8602-9B3525C78497.htm). Learn how to design the finished gradient for land surfaces such as housing subdivisions and retail sites.
*   [Corridor Assembly Tutorials](GUID-2F3011CB-15DC-49E3-BB8B-B04A02B2C12E.htm "These tutorials will get you started working with the corridor assemblies, which create the primary structure of Autodesk Civil 3D corridor models."). Learn how to build and manage assemblies, which are cross sections that are placed incrementally along an alignment.
*   [Corridors Tutorials](GUID-DE41CF86-9A42-4F1F-B880-767F810CE0EB.htm "These tutorials will get you started working with the corridor modeling tools, which are used to design and generate complex carriageway corridor models."). Learn how to create simple and complex corridor designs.
*   [Junction Tutorials](GUID-AAAFC8F1-14E5-4AE4-AE69-BC8DE743AFD4.htm "These tutorials will get you started working with junctions."). Learn how to create complex junctions that dynamically react to changes in the model.
*   [Sections Tutorials](GUID-1801179E-4490-4958-911F-89C038D70DF7.htm "These tutorials will get you started working with sections and section views, which provide a view of the terrain cut at an angle across a linear feature, such as a proposed road."). Learn how to create cross sections of your corridor design, calculate cut and fill earthwork quantities, and create mass-haul diagrams.
*   [Material Calculation Tutorials](GUID-91B964AE-D183-4E50-930C-95C84FA35857.htm "These tutorials will get you started working with the Autodesk Civil 3D tools for calculating and reporting material quantities and volumes."). Learn how to calculate material quantities and generate reports, including rate item reports, earthworks reports, and mass haul diagrams.
*   [Pipe Networks Tutorials](GUID-687FED93-8B35-4BB8-A546-D2E9664339FC.htm "These tutorials will get you started working with the pipe networks features, which you use to design and model the flow and function of a utility system, such as a storm or sanitary sewer."). Learn how to create a pipe network using the specialized layout tools.
*   [Part Builder Tutorials](GUID-99EDD05E-0A51-45BA-B7F8-6F531A2AD791.htm "These tutorials demonstrate how to work with the Part Builder features, which you use to design and model parts (pipes and structures) that are used in pipe networks."). Learn how to design and model parts that are used in pipe networks.
*   [Labels and Tables Tutorials](GUID-BDF3F02E-838E-443D-BFE3-3033F03E214F.htm "These tutorials will get you started creating and editing labels, label styles, and tables."). Learn how to annotate Autodesk Civil 3D objects using labels and tables.
*   [Plan Production Tutorials](GUID-4E042CFC-F82A-4EBF-85E3-B1AE2A0C1C57.htm). Learn how to prepare your design drawings for plotting or publishing.

Each tutorial set contains exercises that are designed to explore the various features of Autodesk Civil 3D.

The tutorial exercises are organized in a logical sequence, based on how you typically work with the different types of features. However, you may complete the exercises in any order you choose. After you begin an exercise, you should complete the steps in the order presented. The first steps provide you with the information you need for the later steps in that exercise.

Note: All drawings used in these tutorials are available in the [tutorials drawings folder](GUID-DECD2305-4906-4329-A973-CFC9B625B4CD.htm#GUID-DECD2305-4906-4329-A973-CFC9B625B4CD__WS27101FEF35E777F9922804FBF5414E08-8000-79) and all source data files are available in the [tutorial folder](GUID-DECD2305-4906-4329-A973-CFC9B625B4CD.htm#GUID-DECD2305-4906-4329-A973-CFC9B625B4CD__WSFACF1429558A55DEF27E5F106B5723EEC-5EEC). If you want to save your changes to the tutorial drawings as you work, save them to the [My Tutorial Data folder](GUID-DECD2305-4906-4329-A973-CFC9B625B4CD.htm#GUID-DECD2305-4906-4329-A973-CFC9B625B4CD__WS1A9193826455F5FF1848157110156D8271-7832) so that you do not overwrite the original tutorial drawings.
