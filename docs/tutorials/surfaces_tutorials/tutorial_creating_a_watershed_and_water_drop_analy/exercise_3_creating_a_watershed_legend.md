# Exercise 3: Creating a Watershed Legend

In this exercise, you will add a watershed legend table to the drawing.

This exercise continues from [Exercise 2: Generating a Watershed Analysis](GUID-00728CAD-1FAB-46CA-BBCC-0728FADEC2DB.htm "In this exercise, you will generate and display the watershed analysis.").

Create a watershed legend table

Note:

This exercise uses _Surface-5A.dwg_ with the modifications you made in the previous exercise, or you can open _Surface-5B.dwg_ from the [tutorials drawings folder](GUID-DECD2305-4906-4329-A973-CFC9B625B4CD.htm#GUID-DECD2305-4906-4329-A973-CFC9B625B4CD__WS27101FEF35E777F9922804FBF5414E08-8000-79).

2.  In Toolspace, on the Settings tab, expand the Surface![](../images/ac.menuaro.gif)Table Styles![](../images/ac.menuaro.gif)Watershed collection.
3.  Right-click the **Standard** style. Click Edit.
4.  In the Table Style dialog box, on the Data Properties tab, clear the Sort Data check box.
5.  Click OK.
6.  Click Annotate tab ![](../images/ac.menuaro.gif)Labels & Tables panel ![](../images/ac.menuaro.gif)Add Tables menu ![](../images/ac.menuaro.gif)Add Surface Legend Table.
7.  At the prompt to specify the table type, enter **Watersheds**.
8.  You are prompted to specify whether you want the table to automatically update if the analysis information changes. Enter **Dynamic** to enable automatic updating.
    
    If a change is made to the surface and the watershed analysis is regenerated, the legend is automatically updated.
    
9.  Click a location in the drawing where you want to place the upper-left corner of the table.
    
    The legend table is displayed in the drawing at the selected location.
