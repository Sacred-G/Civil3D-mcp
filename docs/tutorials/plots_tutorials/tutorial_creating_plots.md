# Tutorial: Creating Plots

This tutorial demonstrates the main methods for creating plots.

You can create plots either from drawing objects, such as lines, arcs, or polylines, or by using the plot layout tools. Plots can also be created automatically by the interaction of alignments and other plots within a site.

Plots are contained in a plots collection, which in turn is contained by a site. Each site can include only one plots collection, but a plots collection can include any number of plots.
