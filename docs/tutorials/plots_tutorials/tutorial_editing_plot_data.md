# Tutorial: Editing Plot Data

This tutorial demonstrates two ways of resizing a plot by moving a plot line.

The tools that you use to edit a plot plot line depends on whether the plot line is attached.

An attached plot line is a plot segment that was created with the slide angle, slide direction, and swing line precise sizing tools on the Plot Layout Tools toolbar. You can use the slide angle, slide direction, and swing line precise editing tools to edit only attached plot lines. In this tutorial, you will learn how to use two precise editing tools:

*   The _Slide Angle – Edit_ tool slides a plot line at a specified angle through a selected plot until a target area is enclosed.
*   The _Swing Line – Edit_ tool swings a plot line from a specified point through a selected plot until a target area is enclosed.

If a plot line is created either from other Autodesk Civil 3D objects or the fixed line and curve tools on the Plot Layout Tools toolbar, it is not attached. You can use grips and the feature line editing tools to change the geometry of these plot lines.
