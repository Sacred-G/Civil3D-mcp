# Tutorial: Displaying and Analyzing Plots

This tutorial demonstrates using plot styles and display order to control the appearance of plots, and exporting reports to analyze plot data.

The plot _display order_, which is set in the Site Plot Properties dialog box, controls the display of shared plot segments. The different styles of plots in a drawing are drawn in a specified order that you can control. Where two styles of plots share a common segment, the displayed color of the segment is determined by the plot style that is drawn last.

Autodesk Civil 3D allows you to export inverse or map check reports for either a single plot, a series of plots, or all plots in a site.

For a tutorial on using the Autodesk Civil 3D survey tools to perform a mapcheck analysis, see the [Outputting Survey Information](GUID-E4B4FC43-4C35-4922-8B56-3447B40FA32C.htm "This tutorial demonstrates how to view information reports for figures and how to use the figures as a source for surface data.").
