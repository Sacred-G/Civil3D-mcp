# Tutorial Folder Locations

Important: Some of the folders noted in this list are subfolders within the **AutoCAD** installation folder (C:\\Programme Files\\Autodesk\\AutoCAD 2025). This is due to the way that AutoCAD-based applications like Autodesk Civil 3D are now installed.

Assemblies folder

_C:\\ProgramData\\Autodesk\\C3D 2025\\enu\\Assemblies\\_

Note: The ProgramData folder may not be displayed by default on your computer. To view this folder, select the option to Show Hidden Files, Folders and Drives on the View tab of the Folder Options dialog box in Windows Explorer.

Autodesk Documents folder

_C:\\Users\\\[user name\]\\Documents\\Autodesk_

Civil 3D Projects folder

_C:\\Users\\Public\\Documents\\Autodesk\\Civil 3D Projects_

Civil 3D Project Templates folder

_C:\\Users\\Public\\Documents\\Autodesk\\Civil 3D ProjectTemplates_

Data folder

_C:\\ProgramData\\Autodesk\\C3D 2025\\enu\\Data\\_

Local Template folder

_C:\\Users\\\[user name\]\\AppData\\Local\\Autodesk\\C3D 2025\\enu\\Template\\_

tutorial folder

_C:\\Programme Files\\Autodesk\\AutoCAD 2025\\C3D\\Help\\Civil Tutorials_

tutorial drawings folder

_C:\\Programme Files\\Autodesk\\AutoCAD 2025\\C3D\\Help\\Civil Tutorials\\Drawings_

My Civil Tutorial Data folder

If this folder does not already exist, create it at the following location to save the tutorial drawings as you work on them:

_C:\\Users\\\[user name\]\\Documents\\Autodesk_
