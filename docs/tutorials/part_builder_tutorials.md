# Part Builder Tutorials

These tutorials demonstrate how to work with the Part Builder features, which you use to design and model parts (pipes and structures) that are used in pipe networks.

Note:

If you have not installed Autodesk Civil 3D to the default location, you may receive messages in the Event Viewer, indicating that pipe network part catalogs are not found. To avoid these messages, it is recommended that you follow the exercises in the order presented. Alternatively, you can reset the path to the catalogs by clicking Home tab ![](../images/ac.menuaro.gif)Create Design panel![](../images/GUID-A7096E57-563B-43D0-B57A-FA69853A76A6.gif)Set Pipe Network Catalog![](../images/GUID-3DCD6FE6-B8D9-4A1F-B32A-AB04E09CE107.png).

Note:

All drawings used in these tutorials are available in the [tutorials drawings folder](GUID-DECD2305-4906-4329-A973-CFC9B625B4CD.htm#GUID-DECD2305-4906-4329-A973-CFC9B625B4CD__WS27101FEF35E777F9922804FBF5414E08-8000-79). If you want to save your work from these tutorials, save the drawings to the [My Tutorial Data folder](GUID-DECD2305-4906-4329-A973-CFC9B625B4CD.htm#GUID-DECD2305-4906-4329-A973-CFC9B625B4CD__WS1A9193826455F5FF1848157110156D8271-7832) so that you do not overwrite the original drawings.
