# Tutorial: Preparing to Annotate a Drawing

This tutorial demonstrates how to perform some optional tasks that can make annotating your drawing easier.

First, you will create external references (Xrefs) to drawings that contain the objects that you will annotate in later Labels and Tables tutorials. Second, you will learn how to use the Autodesk Civil 3D ribbon to locate all of the annotation commands.

**Topics in this section**

*   [Exercise 1: Attaching Drawings as Xrefs for Annotation](GUID-073E6BDC-DFAB-4D34-B3D1-3FB5CA40E54D.htm)  
    In this exercise, you will attach several drawings to one drawing. By attaching drawings, you can annotate multiple large objects in a single compact drawing.
*   [Exercise 2: Exploring the Annotation Tools on the Ribbon](GUID-EB5E85B1-41F6-4336-B0BE-3EB668B2644C.htm)  
    In this exercise, you will learn how to locate annotation tools on the ribbon tabs.
