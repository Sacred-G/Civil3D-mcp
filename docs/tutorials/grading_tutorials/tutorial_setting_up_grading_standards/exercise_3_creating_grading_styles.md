# Exercise 3: Creating Grading Styles

In this exercise, you will create a new grading style and gradient marking.

This exercise continues from [Exercise 2: Creating Grading Criteria](GUID-FDA1CD38-3AE4-48E8-A976-DF2AD16F6611.htm "In this exercise, you will create a grading criteria set for a ditch grading, then create three criteria in the set.").

Create a new grading style

Note:

This exercise uses _Grading-1.dwg_ with the modifications you made in the previous exercise.

2.  In Toolspace, on the Settings tab, right-click the Grading Styles collection. Click New.
3.  In the Grading Style dialog box, click the Information tab. Enter **Ditch** into the name field. Optionally, enter a description for the style.
4.  Click the Display tab and change the color of the Daylight Line to red by clicking its entry in the Color column.
    
    The Select Color dialog box is displayed, where you can select a color. Select red, then click OK.
    
5.  Click the Gradient Markings tab.
6.  Select the Gradient Marking check box. Select the **Basic** style.
7.  Click the style drop down list and click Copy Current Selection.
8.  In the Gradient Marking Style dialog box, on the Information tab, change the name to **Ditch Gradient**.
9.  On the Layout tab, in the Component field, select Component 2.
    
    The parameter table displays values for Component 2, which is the second gradient marking component from the left side. In this case, it is a long line with a tapered line symbol along the top of the gradient.
    
10.  In the Gradient Line Symbol section, change the Symbol Type to None.
     
     **Further exploration:** For Component 1, in the Gradient Line section, change Percent Of Length to a higher value. You can see the results in the Preview pane.
     
11.  Click OK. The new Ditch Gradient pattern is selected for the Ditch grading style.
12.  To view summary information about the style, click the Summary tab.
13.  Click OK.
     
     In Toolspace, on the Settings tab, the Ditch grading style is added to the Grading Styles collection. The Ditch Slope pattern is added to the General![](../images/ac.menuaro.gif)Multipurpose Styles![](../images/ac.menuaro.gif)Slope Pattern Styles collection.
