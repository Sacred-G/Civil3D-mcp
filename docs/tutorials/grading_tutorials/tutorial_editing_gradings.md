# Tutorial: Editing Gradings

This tutorial demonstrates common grading editing tasks, including level adjustment and grading criteria editing.

You use the Grading Level Editor to edit levels along a baseline. A baseline is a feature line or plot line that has had a grading applied to it. You can modify the levels of a plot line unless it is being used as a target for a grading.

You can also insert a VIP (Vertical Intersection Point) between the start and end stations of the baseline, creating an intermediate level point.
