# Tutorial: Creating Gradings

This tutorial demonstrates how to create a feature line and how to gradient from the feature line.

You can create gradings from plot plot lines or _feature lines_. A feature line is a linear object that the grading commands can recognize and use as a baseline. When designing a grading, you must often adjust the levels along a feature line. You create a feature line by converting existing objects, such as lines or polylines.

In the next two exercises, you will perform the following tasks:

*   Create three feature lines.
*   Assign level values along the feature lines using several methods.
*   Create a set of gradings from one of the feature lines.

    In this exercise, you will assign levels to the feature lines you created from AutoCAD lines in the previous exercise.

    In this exercise, you will create a set of gradings, called a grading group, that form a runoff on the side of an embankment.
