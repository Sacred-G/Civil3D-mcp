# Tutorial: Using Surface Profiles

This tutorial demonstrates how to create surface profiles and display them in a profile view.

A surface profile is extracted from a surface, and is often called an _existing ground_ profile. It is displayed on a graph called a _profile view_. Each profile view is associated with a single horizontal alignment, but it can display multiple surfaces and offset profiles from that alignment. An _offset profile_ displays surface levels at a specified distance horizontally offset from the alignment. When you create a profile along the centerline of a horizontal alignment, Autodesk Civil 3D can create one or more offset profiles automatically.

Surface profiles can be either static or dynamic. A _static_ profile shows the levels at the time it was created, but does not react to later changes in the alignment or surface. A _dynamic_ profile automatically changes if the level changes along the horizontal alignment, so it remains current.
