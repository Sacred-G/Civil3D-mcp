# Exercise 1: Creating Offset Alignments

In this exercise, you will create dynamic offset alignments for an existing centerline alignment.

Create offsets of a centerline alignment

1.  Open _Align-6A.dwg_, which is located in the [tutorials drawings folder](GUID-DECD2305-4906-4329-A973-CFC9B625B4CD.htm#GUID-DECD2305-4906-4329-A973-CFC9B625B4CD__WS27101FEF35E777F9922804FBF5414E08-8000-79).
    
    The drawing contains a surface and an alignment.
    
2.  Click Home tab ![](../images/ac.menuaro.gif)Create Design panel ![](../images/ac.menuaro.gif)Alignment drop-down ![](../images/ac.menuaro.gif)Create Offset Alignment ![](../images/GUID-88B05EDA-DFA3-4010-89FE-924C5DA869D3.png) Find.
3.  Select the alignment in the drawing.
    
    ![](../images/GUID-DFDC3A48-37DF-414C-8697-507492A8BB79.png)
    
4.  In the Create Offset Alignments dialog box, specify the following parameters:
    
    *   No. Of Offsets On Left: **1**
    *   No. Of Offsets On Right: **1**
    *   Incremental Offset On Left: **12**
    *   Incremental Offset On Right: **12**
    *   Alignment Style: **EC**
    *   Alignment Label Set: **Offset Labels**
    
    Note: For more information about the other options in this dialog box, including options for adding widening and creating offset profiles, see [Create Offset Alignments Dialog Box](https://beehive.autodesk.com/community/service/rest/cloudhelp/resource/cloudhelpchannel/guidcrossbook/?v=2025&p=CIV3D&l=ENG&accessmode=live&guid=GUID-3D008CBA-8C57-42D9-ADB9-EB6BE585281C).
    
5.  Click OK.
    
    An offset alignment is created on either side of the centerline alignment.
    
    ![](../images/GUID-BF69FD83-91D6-4654-B422-B86F7772E0AE.png)
    
6.  In Toolspace, on the Prospector tab, expand the ![](../images/GUID-A1AF1DE4-7370-4208-8B4C-ABB3CB562C79.png)Alignments![](../images/ac.menuaro.gif)![](../images/GUID-C84B1C0E-2655-4C96-9DAB-455672773ADC.png)Offset Alignments collection.
    
    Notice that the two alignments you created were placed in the Offset Alignments collection. You can create profiles, profile views, and sample line groups from offset alignments, just as you can for standard, Centerline alignments.
