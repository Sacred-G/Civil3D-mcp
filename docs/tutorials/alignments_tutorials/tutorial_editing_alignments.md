# Tutorial: Editing Alignments

This tutorial demonstrates some common editing tasks for alignments.

Editing the layout parameter values of an alignment allows you to make fine adjustments to alignment sub-elements. **Grip editing** provides a convenient method of manually reshaping an alignment.

**Topics in this section**

*   [Exercise 1: Editing the Layout Parameter Values of an Alignment](GUID-70269D00-79DF-4FB1-B945-FB5A107C11DE.htm)  
    In this exercise, you will use the Alignment Elements vista and Alignment Layout Parameters dialog box to edit the layout parameter values of an alignment.
*   [Exercise 2: Grip Editing an Alignment](GUID-2B7CE63E-F2F6-4C54-AB23-C84491FFB5F4.htm)  
    In this exercise, you will use grips to move alignment curves.
*   [Exercise 3: Applying a Mask to an Alignment](GUID-B4208BE1-BF5E-41DB-922A-E08EE62AD5C7.htm)  
    In this exercise, you will hide a portion of an alignment from view.
